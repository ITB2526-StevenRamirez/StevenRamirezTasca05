MusicIAn Project

Developed in 2025

General Description

MusicIAn was a school project whose main idea was to implement audio tools in a web environment. The challenge was to create audio manipulation tools using programming languages in which we had no prior experience, as well as implementing an AI.

Developed Tools

  • Track Separator: This tool separates the audio file into 4 different tracks (vocals, bass, drums, others) using Spleeter AI.
  • Unifier: Allows combining two audio files into a single one. Developed entirely in JavaScript.
